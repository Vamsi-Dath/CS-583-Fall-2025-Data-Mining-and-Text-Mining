# CS-583-Fall-2025-Data-Mining-and-Text-Mining
Research project - Sentiment Analysis

# Folder structure
```
# pipeline
data_cleaning.ipynb -> prediction pipeline

# train dataset
requirements.txt -> process dependencies
data/train -> training dataset

# classification model
logistic_regression_model.pkl -> trained model

# test dataset
final-testData-no-label-Obama-tweets.xlsx -> final Obama test dataset
final-testData-no-label-Romney-tweets.xlsx -> final Romney test dataset

# final evaluation files
final_obama_predictions.txt
final_romney_predictions.txt

```
